# Changelog

All notable changes to this project will be documented in this file.

## [0.0.0.2] - 2026-05-11

### Added
- Interactive VPC visualizer — generates HTML visualization of VPC subnets
- CIDR map layout with subnet bars positioned by CIDR range
- Dark theme with public/private color coding
- Hover tooltips showing subnet details (CIDR, type, AZ, IPs)
- Click-to-expand subnet bars
- Zoom controls (in, out, reset)
- Multi-VPC dropdown switching
- Subnet details panel
- `--visual` flag to generate and open HTML visualization
- `--multi-vpc` flag to visualize all VPCs in the region
- Test suite for visualizer module
